## DeepVi: AI 기반 비전 분석 플랫폼

DeepVi는 사전 정의된 자동화 워크플로우를 통해 실시간 라벨링, 세그멘테이션, 모델 학습 및 배포를 간소화하는 엔드투엔드 AI 비전 플랫폼입니다. 데이터 수집, AI 기반 분석, 결과 처리를 단일 플랫폼으로 통합하여 최소한의 수작업으로 원시 이미지와 영상을 실행 가능한 인사이트로 변환합니다.

## DeepVi를 선택하는 이유

DeepVi로 할 수 있는 것:
- 고품질 학습 데이터 준비를 위해 이미지와 영상을 라벨링하고 세그멘테이션합니다.
- 커스텀 머신러닝 모델을 쉽게 학습하고 배포합니다.
- 간단한 드래그 앤 드롭 인터페이스를 사용하여 데이터 수집부터 분석 및 실행까지 전체 워크플로우를 자동화합니다.

기존 도구와 달리 DeepVi는 실시간으로 워크플로우를 실행하여 더 빠른 결과와 원활한 운영을 제공합니다.

## 주요 기능

### 데이터 소스 관리
- 스토리지 관리
폴더를 정리하고, 스토리지 용량을 모니터링하며, 원시 데이터와 학습 데이터셋을 원활하게 분리합니다.
- 업로드 기능
로컬 기기에서 중앙 데이터 저장소로 이미지와 영상을 손쉽게 업로드합니다.

### 데이터 큐레이션
- 라벨링 & 세그멘테이션
정확한 학습 데이터셋을 만들기 위해 이미지와 영상을 정밀하게 어노테이션합니다.
- 학습 데이터셋 생성
라벨링된 이미지를 모델 학습에 적합한 구조화된 데이터셋으로 변환합니다.

### 머신러닝 모델 관리
- 모델 학습 & 테스트
라벨링된 데이터셋으로 ML 모델을 학습하고 검증합니다.
- 모델 데이터베이스 & 배포
단 몇 번의 클릭으로 학습된 모델을 버전 관리, 관리 및 배포합니다.

### 워크플로우 구성
- 엔드투엔드 자동화
데이터 수집, AI 분석, 결과 처리를 아우르는 커스텀 워크플로우를 설계합니다.
- 드래그 앤 드롭 GUI
코딩 없이 시각적으로 워크플로우를 구성합니다.

### 워크플로우 노드 관리
- 데이터 소스 노드
데이터 수집기를 등록하고 설정하여 데이터 수집 프로세스를 시작합니다.
- 데이터 싱크 노드
시각화, 데이터베이스 업데이트, 알림, MES 연동 등 분석 결과를 처리하고 실행합니다.

### 배포 & 모니터링
- 워크플로우 제어
완전한 유연성을 위해 언제든지 워크플로우를 시작, 일시정지 또는 중지합니다.
- 실시간 모니터링
원활한 실행을 보장하기 위해 워크플로우 진행 상황과 노드 출력을 실시간으로 모니터링합니다.
